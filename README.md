# Build with Bigdata.com — Claude Plugin

A Claude plugin that helps developers integrate with [Bigdata.com](https://bigdata.com) Service APIs. It provides guided assistance for working with the full suite of Bigdata.com endpoints, from search and structured data to workflows and the research agent.

## Getting Started

1. **Install the plugin**: Add `build-with-bigdata-com` to your Claude environment.
2. **Get an API key**: Sign up at [Bigdata.com](https://bigdata.com) and [generate an API key](https://platform.bigdata.com/api-keys).
3. **Explore the APIs**: Browse the [API Reference](https://docs.bigdata.com/api-reference/workflows/execute-workflow) for endpoint details, request/response schemas, and authentication requirements.
4. **Try the playgrounds**: The [Developer Platform](https://platform.bigdata.com/) includes interactive playgrounds with example requests and responses for every API.

## What's Included

| Component | Description |
|-----------|-------------|
| `skills/build-with-bigdata/` | Skill with API reference material covering Search, Volume, Knowledge Graph, Co-mentions, Batch Search, and more. |
| `commands/` | Slash commands for common integration tasks. |
| `tests/` | Use-case test scripts and prompt examples for validating plugin behavior against real API workflows. |
| `.claude-plugin/plugin.json` | Plugin manifest. |
